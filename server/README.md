# Server (optional)

The main app runs **without** this server. The Next.js app in `web/` uses **Firebase** (Auth, Firestore, Storage) directly from the browser.

This folder is kept only for reference or for adding a thin API later (e.g. webhooks, server-only logic). You do **not** need to run it for normal use.

See **docs/ARCHITECTURE.md** in the project root for the current architecture.
